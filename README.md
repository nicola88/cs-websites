# Computer Science websites

Snaphots of computer science websites for offline viewing.

```bash
wget --reject pdf \
     --recursive \
     --no-clobber \
     --page-requisites \
     --html-extension \
     --convert-links \
     --restrict-file-names=windows \
     --domains site_domain> \
     --no-parent site_url
```
